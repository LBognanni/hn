# HN Reader

A **very** minimal Hacker News reader built for mobile. Install it to your home screen and browse the day's top stories without the clutter.

View it live:

### 👉️ https://lbognanni.github.io/hn/ 👈️

## Features
- View the top stories of the day, sorted by number of comments
- Clicking on a story opens the discussion
- Comments and threads are collapsible
- It remembers which stories you've read, and which comment threads you've collapsed
- Works offline with cached data from the last successful fetch
- **Tiny and fast!**
